由生病想到生命

| |

我属鸡。在20多岁的年纪，我的感觉和王小波说的一样，是生命的黄金时代，我觉得我能够一直生猛下去，没想到生活就是个缓慢受锤的过程。我最近正在经受一次感冒之锤。

我平日也经常锻炼，应该挺强壮，这几天却因为感冒要带一堆纸巾，要穿秋裤。病来如山倒啊，就像你在田野中守望麦田，盼望稻香的时候，大雨瓢泼而下，避无可避，能做的只是在雨滴的捶打中快点回家。就在我已经快完了感冒是什么样的体验的时候感冒及时地登门拜访，带我重温旧梦。可不要小瞧感冒，也不要笑话我虚，毕竟当年西班牙流感吞噬的性命比一战二战死亡总人数还要多。不过我我患的感冒应该只会吞噬一些白细胞，但它除了给我带来了身体上一些不适，还给我在思想上带来了一些困扰。

帕斯卡尔说人是会思想的芦苇。我觉得会思想是比会使用工具还要大的进步。当其他生物的生命主题围绕在生存、繁殖的问题上时，人类可以“杞人忧天”般去寻找背后的意义。思考在受锤时犹为强烈：所有的失恋青年摇身一变都成了爱情哲学家，落魄的失败者总有更多的人生感悟，唐僧更是在九九八十一锤中思考到死去若干脑细胞，头发都没有了。我上次崴脚后，颤颤巍巍地站在场边，脑子一团乱麻。我在想为什么受伤的总是我，为什么来了西安就老受伤，为什么偏偏在去城墙看灯的前一天崴脚。就不得不怀疑，难道这里真的是我的失意之地、痛苦之源吗？我曾经跨过大山大海，但是山海不可平，这时候会有两种选择：一个是像漂流在海上的少年派，在万分痛苦中开始怀疑上帝，为什么那么信仰他他却不能保佑自己；一种是像饱受黑死病折磨的欧洲人民，以为是上帝在惩罚自己，主动用铁鞭抽打自己，以为就可以躲避黑死病的惩罚。

人类的认识总是有限的，当我们的知识不能解释的时候，就出现了宗教或者迷信。其实这也是解释问题的一种方法。神这种东西，不好证实，更不容易证伪，一般人宁可信其有，不可信其无。中国人大多是无神论，但中国人经常挂在嘴边的是命。命中缺什么，就在名字里放个什么字。五行都缺，就叫金灶沐。喜欢的一档综艺《极限挑战》的口号就是“这就是命”，可见信命还不能称之为宗教，不然广电总局不会不管的。正如每期节目的剧情，我们先是要努力，斗智斗勇，至于结果，多少还是会有随机因素作祟，我们坦然接受，把那部分称之为命运。所以说，祁同伟所向往的“胜天半子”是不可能实现的，因为本来没有所谓的天、命，是我们在可能面前束手无策的时候才创造了天命来解释之。所谓的信不信命，其实就是愿不愿意用这么一套方法来解释问题。从不相信到相信的过程，就是一个受锤的过程，在这个过程中，可能变消极了，也可能变得超然，得到解脱。

我刚才又读了一遍史铁生的《我与地坛》。他常年在地坛中静坐，遇到各种各样的人，见识了不同的命运。有一个朋友，连续几年参加环城跑，当他跑了十五名时，新闻橱窗中挂了前十名的照片；当他跑了第四，却只挂了前三名的照片，当他终于成为第一，橱窗挂的是航拍的群众大镜头。他绝望，他破口大骂。但后来当他最后一次参加赛跑获得第一，有田径教练说后悔没有早一点发现他的时候，他只是笑了笑，什么也没有说，后来和史铁生说起，也还是很平静。经历得越多，被锤得越久，就越超然。有些结，注定是解不开的，我们只能试着去看开。

感冒到生日这天还是没好，难道本命年的坎坷真的尤其多吗？思考了上面这么多，在命运的锤下，我默默穿上了过年买的花花公子红内裤。。
